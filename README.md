# Mason Navigator — Kiosk

## Setup
```
npm install
npm start          # dev mode (windowed)
npm run build-win  # → dist/Mason Navigator Setup 1.0.0.exe
```

## First run
1. Run `npm start` to test in windowed mode
2. Click ⚙ to set the kiosk's "You Are Here" room and name
3. When ready for deployment: `npm run build-win`

## Controls
- Ctrl+Shift+D  → DevTools
- Ctrl+Q        → Quit
